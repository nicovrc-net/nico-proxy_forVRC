# nico-proxy_forVRC
[VRChat向けニコニコ動画再生支援ツール](https://nicovrc.net/)の動作プログラム
## 必要なもの
- Java 21以降
## あれば嬉しいもの
- HTTPS接続ができるHTTP Proxy
- Redis

## このプロジェクトのbranch
(適当に運用なのでたまーにmasterに直コミットする場合あり)
- master : nicovrc.netに適用時に下記記載のブランチからmergeされる
- v0     : ver 0.x (開発終了)
- v1     : ver 1.x (開発終了)
- v2     : ver 2.x (開発終了)
- v3     : ver 3.x (開発中ブランチ)
